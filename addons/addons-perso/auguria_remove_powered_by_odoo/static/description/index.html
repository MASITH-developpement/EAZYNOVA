How to Use

Prerequisites: Enable Developer Mode to access the Technical menu. Go to Settings → Activate Developer Mode.

Steps to follow:

  1. Go to Settings → Technical → Email Templates
  2. Select all templates (or only the ones you want to modify)
  3. Click Action → Remove Powered By Odoo

Multilingual Note

If you use Odoo in multiple languages, repeat these instructions for each language to remove "Powered By Odoo" from all translated templates.

Tip: Make sure to switch languages in Odoo settings and apply the removal action for each language to ensure complete branding removal across your entire system.
